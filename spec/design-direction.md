# Design Direction — Chez Violeta Intelligence

## Visão Geral
Interface analítica clean e profissional para operações de joalheria.
Foco em usabilidade para tomadores de decisão (não analistas técnicos).
Prioridade: KPIs operacionais (vendas, estoque, margem) com drill-down
até nível SKU. Visual storytelling com hierarquia clara.

## Princípios Macro
1. **Data-First, Decoration-Last**
   Dashboard funcional antes de polimento visual.
   Toda tela começa por um número que responde "qual a pergunta?".

2. **Consistência Cross-Dashboard**
   Mesma paleta, mesma tipografia, mesma hierarquia de cards
   em vendas, estoque e financeiro. Design system único.

3. **Mobile-Aware, Desktop-First**
   Layout otimizado para 1920x1080, mas legível em tablets.
   Evitar scroll horizontal.

4. **Acessibilidade BG-A**
   Contraste WCAG 2.1 AA. Ordem de leitura natural (esquerda→direita, topo→base).
   Não usar cor como único canal de informação.
